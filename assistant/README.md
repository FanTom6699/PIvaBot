# Mafiozi Assistant

Асинхронный помощник для Telegram-игры через пользовательский аккаунт Telethon.

Целевой чат по умолчанию: `@moolokobot`.

## Что умеет

- авторизуется через Telethon user account;
- открывает диалог только с целевым ботом;
- читает входящие и отредактированные сообщения;
- сохраняет текст, inline-кнопки, ресурсы, задания, животных и таймеры в SQLite;
- передаёт состояние, `recipes.json` и `rules.json` в LLM;
- принимает от LLM только JSON-действия: `click`, `open`, `wait`, `message`, `noop`;
- нажимает inline-кнопки и открывает меню;
- scheduler следит за готовыми таймерами;
- OpenAI вынесен в отдельный адаптер, ядро можно подключить к другой LLM.

## Запуск

1. Получи `TG_API_ID` и `TG_API_HASH` на `https://my.telegram.org`.
2. Скопируй пример конфига:

```bash
cp assistant/.env.example .env
```

3. Заполни `.env`.
4. Установи зависимости:

```bash
pip install -r requirements.txt
```

5. Первый запуск лучше делать в безопасном режиме:

```bash
python -m assistant.main
```

Пока `DRY_RUN=true`, помощник будет думать и логировать действия, но не будет нажимать кнопки.
Когда проверишь логику, поставь:

```env
DRY_RUN=false
```

## Важно

Помощник специально ограничен одним диалогом `TARGET_BOT`. Он не читает и не выполняет действия в других чатах.
